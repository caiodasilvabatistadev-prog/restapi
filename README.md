Pizzaria Back-End API | FastAPI & SQLite
Este repositório contém o código-fonte do back-end para uma aplicação de pizzaria. A API foi desenvolvida utilizando o framework FastAPI, conhecido por seu alto desempenho e facilidade de uso com type hints do Python. O SQLite foi escolhido como banco de dados para simplificar a configuração e o desenvolvimento local.
O projeto implementa funcionalidades essenciais para o gerenciamento de uma pizzaria, incluindo operações CRUD para:
Gerenciamento de clientes
Cardápio (pizzas, bebidas, etc.)
Processamento e rastreamento de pedidos
Tecnologias Principais:
Python: Linguagem de programação principal
FastAPI: Framework web para construção da API RESTful
SQLite: Banco de dados relacional leve
SQLAlchemy (ou outro ORM, se aplicável): Para interagir com o banco de dados
Uvicorn: Servidor web ASGI para rodar a aplicação
O objetivo deste projeto é demonstrar a implementação de uma API robusta e moderna, seguindo as melhores práticas de desenvolvimento, como a arquitetura de camadas e operações CRUD eficientes.
